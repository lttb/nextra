# Dropdown

The `Dropdown` component is used to display a list of options.

```tsx
import { Dropdown } from '@components/Dropdown'

export default (
  <Dropdown
    label="Dropdown"
    options={[
      { label: 'Option 1', value: 'option-1' },
      { label: 'Option 2', value: 'option-2' },
      { label: 'Option 3', value: 'option-3' }
    ]}
  />
)
```
